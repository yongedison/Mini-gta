# 🎆 Neon World Experiment

A visual experiment focusing on advanced CSS effects, glow styling, and interactive map layouts.

### 🧪 Features
* **Neon Aesthetics:** High-contrast visuals with modern lighting effects.
* **Interaction Testing:** Experimenting with player movement and collision logic on mobile.
* **UI/UX Focus:** Learning how to build immersive environments using only HTML and CSS.

### 🚀 Development Note
This project is an ongoing experiment to master mobile-based game development and interactive UI.
